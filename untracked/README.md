# Untracked — Development Reference

This directory contains documents that were used during development but are
not part of the user-facing documentation. They're preserved for reference
but not maintained as current sources of truth.

## Contents

- **Security audits**: Platform, OpenCode runtime, and Pi runtime security audits
- **Plans**: Security hardening plan, architecture diagram redesign spec
- **Runtime references**: Upstream reference documentation for OpenCode, Mistral Vibe, and Pi-Mono runtimes
- **Community & program**: Contributor program and community guidelines (drafts)
- **Reports**: Accessibility conformance report

## Note

These documents may contain outdated claims, plans that were never implemented,
or findings that have since been addressed. For current information, see the
[docs/](../docs/) directory and the [README.md](../README.md).
