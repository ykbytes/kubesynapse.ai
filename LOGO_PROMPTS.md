# KubeSynthAI Logo Generation Prompt

## Project Overview
**KubeSynthAI** is a Kubernetes-native AI agent orchestration platform. It enables enterprises to build, deploy, and manage AI agents with unified control planes, multiple runtime support (LangGraph, Goose, Codex, OpenCode), file-backed skills, agent-to-agent delegation, and enterprise-grade security—all running natively inside Kubernetes clusters.

**Key themes**: Kubernetes orchestration, AI agents, enterprise automation, cloud-native, multi-agent coordination, secure execution, developer productivity.

## Brand Guidelines Summary

- **Primary Colors**: Kubernetes Blue `#326CE5`, AI Purple `#7C3AED`, Cyan `#06B6D4`
- **Secondary Colors**: Deep Navy `#1E3A8A`, Electric Blue `#3B82F6`, Violet `#8B5CF6`, Teal `#2DD4BF`
- **Typography**: Modern geometric sans-serif (Inter, SF Pro, Roboto, or similar)
- **Style**: Flat vector, minimalist, subtle gradients, clean lines
- **Avoid**: Overly complex details, cartoonish elements, heavy 3D rendering, cluttered compositions
- **Aspect Ratios**: Prepare both horizontal (icon + text) and square/icon-only variants

## AI Image Generator Tips

When using these prompts with tools like **Midjourney**, **DALL-E 3**, **Stable Diffusion**, or **Adobe Firefly**:

1. **Add aspect ratio flags** if needed:
   - Midjourney: append `--ar 16:9` or `--ar 1:1`
   - DALL-E: specify "wide banner" or "square icon" in the prompt

2. **For vector output**, add: *"vector art style, clean lines, SVG-ready, flat design"*

3. **For higher fidelity**, append: *"professional logo design, scalable, no text artifacts, suitable for tech startup branding"*

4. **If results are too busy**, add: *"minimalist, lots of white space, simple geometric shapes only"*
