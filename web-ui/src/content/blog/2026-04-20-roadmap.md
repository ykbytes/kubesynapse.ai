KubeSynapse is under active development. Here is an honest look at where we are headed and what's on the roadmap.

## Near-Term (Next 4 Weeks)

### Mistral Vibe Runtime

A new `runtime.kind: mistral-vibe` for vision-capable agents. Mistral Vibe can process images and screenshots, enabling agents that can visually inspect dashboards, verify UI changes, or analyze infrastructure diagrams.

### Production Helm Hardening

- Liveness and readiness probes on all components
- Pod Disruption Budgets for graceful upgrades
- Resource requests and limits tuned for real workloads
- Structured JSON logging across all services
- Database connection pooling and index optimization

### Enhanced Network Policies

Tighter pod-to-pod communication rules. Worker jobs get explicit egress policies. Agent pods are isolated by default with controlled access to shared services.

## Mid-Term (1-3 Months)

### A2A Cross-Cluster Communication

Agents in different clusters can invoke each other through the Agent-to-Agent JSON-RPC protocol. Useful for multi-region deployments where a central coordination agent delegates to regional execution agents.

### Context Bundle Management

Upload and manage shared knowledge bases that agents can reference. Documents, runbooks, and configuration references stored as Kubernetes resources with version tracking.

### Expanded MCP Catalog

New sidecars planned:

- **Database** — PostgreSQL, MySQL, MongoDB query tools
- **Cloud Providers** — AWS, GCP, Azure resource management
- **Monitoring** — Prometheus, Grafana, Datadog integration
- **CI/CD** — GitHub Actions, ArgoCD, Flux interaction

### Evaluation Framework

Repeatable quality testing for agents. Define evaluation suites with input scenarios and expected outputs. Run them on every agent change to catch regressions. Track metrics across runs.

## Long-Term (3-6 Months)

### Multi-Tenancy Enhancements

Full namespace isolation with per-tenant quotas, billing, and access control. AgentTenant CRDs define resource boundaries and policy inheritance.

### GitOps Integration

First-class ArgoCD and Flux support. Agent definitions tracked in Git, reconciled automatically, with drift detection and rollback.

### Agent Marketplace

A curated library of pre-built agent templates and workflow patterns. Deploy a security scanner, capacity planner, or deployment automation with one click.

## How to Influence the Roadmap

This is an open-source project. The roadmap is shaped by the community:

- Open an issue on GitHub to request features
- Join the Discussions forum for design conversations
- Submit PRs — we review and merge contributions regularly

We prioritize based on production utility, security impact, and community demand.
